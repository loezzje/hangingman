Hangman

A basic game to practise React/reducer. 

### to play locally:

Make sure to have [Node JS](https://nodejs.org/en/download/) and [React](https://facebook.github.io/react-native/docs/getting-started.html)

```bash
git clone git@github.com:loezzje/hangingman.git
cd hangingman
npm install
npm start
```
